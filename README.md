# ApplianceHub - Online Appliance Shopping Platform

A full-featured e-commerce platform for buying home appliances, built with FastAPI, React, and MongoDB.

## Features

### Customer Features
- **User Authentication**: JWT-based secure authentication with email/password
- **Product Catalog**: Browse premium appliances by category
- **Search & Filter**: Find products by name, brand, or category
- **Shopping Cart**: Add/remove items, adjust quantities
- **Secure Checkout**: Stripe payment integration
- **Order History**: View all past orders with status tracking
- **Product Reviews**: Rate and review purchased products
- **AI Recommendations**: Get personalized product suggestions based on purchase history

### Admin Features
- **Dashboard**: View revenue, order statistics
- **Product Management**: Create, update, delete products
- **Order Management**: Update order status (pending, processing, shipped, delivered)
- **Inventory Control**: Track product stock levels

## Tech Stack

Backend: FastAPI + MongoDB
Frontend: React + Tailwind CSS + Shadcn/UI
Payments: Stripe
AI: OpenAI GPT-4 (via Emergent LLM Key)
